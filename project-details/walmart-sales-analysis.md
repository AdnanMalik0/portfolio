# Walmart Sales and Customer Analysis

Source GitHub Repository: <https://github.com/AdnanMalik0/Analysing-Walmart-Sales-its-Customers>

## Snapshot

This project uses Tableau to analyse Walmart sales and customer behaviour from 2012 to 2016. The dashboard focuses on both overall sales performance and how customer segments differ in their purchasing behaviour.

## Goal

The goal was to understand Walmart sales patterns and customer-segment behaviour, then communicate those insights through an interactive Tableau dashboard.

## What Was Built

- A Tableau dashboard using Walmart product-sales and customer data
- Sales trend analysis for 2012 to 2016
- Customer-segment analysis to compare purchasing behaviour across segments
- A published Tableau Public dashboard for interactive review

## Evidence Included

- A completed Tableau workbook for interactive analysis
- The supporting Walmart sales and customer dataset
- Dashboard visuals showing the final reporting output

## Live Dashboard

Tableau Public dashboard:

<https://public.tableau.com/app/profile/adnan6443/viz/WalmartSalesitsCustomersAnalysis/Dashboard1>

## Result / Outcome

The dashboard provided useful insight into how different customer segments behave and how they can be targeted more effectively. It also demonstrated how sales and customer views can be combined in one interactive BI asset.

## Portfolio Takeaway

This project adds strong visualisation breadth to the portfolio by showing Tableau-based storytelling and customer-segmentation analysis in an interactive format.
