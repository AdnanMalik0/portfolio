# Effects of Gender Distribution on School Performance

Source GitHub Repository: <https://github.com/AdnanMalik0/Analysing-Effects-of-Gender-Distribution-in-School-Performance>

## Snapshot

This project analysed schools in England to assess whether gender distribution and school type are associated with performance outcomes. It combined public education data with statistical interpretation around school composition and pupil achievement.

## Goal

The objective was to examine whether the number or proportion of boys and girls in a school has an observable relationship with school performance, using OFSTED ratings and pupil outcomes in reading, writing, and mathematics.

## What Was Built

- A Python analysis workflow using public school-performance data
- A comparison across mixed schools and single-sex schools
- Performance analysis using OFSTED ratings and pupil outcomes
- Interpretation of how school gender composition relates to overall results

## Data Sources

The analysis used UK government school-performance data including:

- school information and school type
- OFSTED rating information
- reading, writing, and mathematics performance measures
- male and female pupil counts and proportions

## Evidence Included

- A full analysis notebook covering the statistical workflow
- A written report explaining the interpretation and conclusions
- Supporting public education datasets describing school type, pupil mix, and performance

## Result / Finding

The analysis found that mixed schools were the majority, girls-only and boys-only schools were split roughly evenly, girls-only schools appeared to perform better than boys-only schools, and schools with a higher percentage of girls tended to perform better overall.

## Portfolio Takeaway

This project is strong because it combines public-sector data analysis, education-policy interpretation, and statistical reasoning on a dataset with clear real-world relevance.
